# PomoFox releases

pixel pomodoro fox for the macOS menu bar — download the latest DMG from
[Releases](https://github.com/xrf9268-hue/pomofox-releases/releases).

## install

1. download `PomoFox-x.y.z.dmg`, drag PomoFox into Applications.
2. first launch on macOS 15+: the app is adhoc-signed, so Gatekeeper will block it once.
   open **System Settings → Privacy & Security**, scroll down, click **Open Anyway**.
   (Control-click → Open no longer bypasses Gatekeeper since macOS Sequoia.)
3. after that, updates arrive in-app automatically (Sparkle, EdDSA-signed feed:
   [`appcast.xml`](./appcast.xml)).

apple silicon only · macOS 15+ · personal project, source not published.
